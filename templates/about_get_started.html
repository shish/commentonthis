Getting Started with Comment on This!

First you need to create an account, the standard "pick a username / password" deal.

Then, Insert this code into your web pages to start accepting comments:

<script src="http://www.commentonthis.net/cot/cot.js"></script>
<script>cot_init("YourUsername");</script>

By default the script will attach to all P tags - for more control, you can pass a second paramater which will be used as a jQuery selector; for example, you could take comments on any paragraph with an explicit ID: 

<h1>My Page</h1>
<p>~~~ decoration! ~~~</p>
<p id="worthy">Here's a paragraph worthy of being commented on</p>
<p>* sparkles! *</p>
<p id="meaning">Some more meaningful, non-decoration text</p>
<p>-----------</p>

<script src="http://www.commentonthis.net/cot/cot.js"></script>
<script>cot_init("YourUsername", "P[id]");</script>

Or to get comments on paragraphs within a given section:

<h1>My Page</h1>

<section class="description">
    <p>This is the documentation for Foo</p>
</section>

<section class="doc">
    <p>You can frob the bar with a wibble-stick.</p>
    <p>You can wibble the qux with some WD40.</p>
    <p>Greasing the badger-wheel is recommended.</p>
</section>

<script src="http://www.commentonthis.net/cot/cot.js"></script>
<script>cot_init("YourUsername", ".doc P");</script>

You don't even have to hook into P tags - you could collect comments on images, or sections, for example.
