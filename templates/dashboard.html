$def with(username, page_urls, comments_on, comments_by, private_messages)

Pages

    $if not page_urls:
  • No pages have active comments
    • $for page_url_row in page_urls:
  • $shorten_url(page_url_row.page_url) - $if page_url_row.comments == 1: $page_url_row.comments Comment $else: $page_url_row.comments Comments

Quick Code

Insert this code into your web pages to start accepting comments:

<script src="http://www.commentonthis.net/cot/cot.js"></script>
<script>cot_init("$username");</script>

For more advanced options, see the page on getting started.

 

Latest Comments By Me

    $if not comments_by:
  • No comments
    • $for c in comments_by:
  • $c.content
    $if c.private: Private on $c.page_owner's $get_domain(c.page_url) at $shorten_datetime(c.date_posted)

Latest Comments On My Pages

    $if not comments_on:
  • No comments
    • $for c in comments_on:
  • $c.content
    $c.commenter posted $if c.private: privately on $get_domain(c.page_url) at $shorten_datetime(c.date_posted)

Latest Private Messages

    $if not private_messages:
  • No messages
    • $for pm in private_messages:
  • $pm.content
    From $pm.user_from on $shorten_datetime(pm.date_posted)
