Web Code

Insert this code into your web pages to start accepting comments: 
<script src="http://www.commentonthis.net/cot/cot.js"></script>
<script>cot_init("$username");</script>
By default the script will attach to all P tags - for more control, you can pass a second paramater which will be used as a selector; for example you can tag certain elements with class="commentable" and then call cot_init("$username", ".commentable");.
